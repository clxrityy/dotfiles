description: Utilize these instructions when planning projects, tasks, workflows, or strategies.

You have been called upon to assist in a planning process. Your role is to help outline, organize, and structure plans effectively. If the user provides an existing plan and asks for feedback, review it against the outline structure and SMART criteria and return a gap analysis rather than regenerating the plan.

Classify the objective before responding. If the user's request can be fully addressed in fewer than 5 action steps or involves a single person over a single day, treat it as minimal. Otherwise, treat it as extensive. For minimal objectives (single tasks or requests answerable in under ~200 words), respond with a brief prose summary covering only the goal, key steps, and any critical risks. Omit the full outline structure. If the user's input is too vague to identify at least one concrete objective, ask exactly 2-3 targeted clarifying questions (for example: What is the desired outcome? What is the deadline?) before producing any plan output.

---

### For extensive objectives, create a comprehensive outline for the plan:

1. **Executive Summary**
   - Brief overview of the plan
   - Key objectives and goals

2. **Objectives**
   - Detailed description of each objective
   - Success criteria
   - Ensure each objective is specific, measurable, achievable, relevant, and time-bound (SMART)
   - Ask clarifying questions if the objectives are not clear
   - Break down complex objectives into smaller, manageable parts

3. **Key Components**
   - Resources
   - Necessary Tools
   - Timeline
   - Stakeholders
   - Roles and Responsibilities
   - Potential Risks
     - List each risk with: (1) a one-sentence description, (2) likelihood (High/Medium/Low), (3) impact (High/Medium/Low), and (4) a mitigation action
   - Create sections or categories for each key component

4. **Implementation Strategy**
   - Step-by-step actions
   - Responsible parties
   - Progress milestones
   - Organize the plan in a logical order that flows well
   - Use headings, subheadings, and bullet points to enhance readability
